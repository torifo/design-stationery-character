# POCO POCO — design-stationery-character

> **"Make study a little fun, every day."**

A design study of a simple, student-targeted stationery shop, executed under the **character** theme — featuring three original mascots (Shizuku / Mame / Fuku-chan).

## Overview

| | |
|---|---|
| **Brand** | POCO POCO |
| **Theme** | Character / Kawaii |
| **Target** | Elementary & middle-school students |
| **Live Site (planned)** | `design.stationery-character.riumu.net` |

## Design Concept

- **Color**: Pink `#FFB3C6` × yellow `#FFE566` × pastel blue / green × ink-grape `#3A2B3E`
- **Typography**: Zen Maru Gothic + M PLUS Rounded 1c
- **Aesthetic**: Rounded shapes, sticker-style hard shadows, dotted borders
- **UX**: 6 items, 3 mascot profiles, news strip, yellow info card

## Tech

- Pure HTML + CSS Custom Properties + Vanilla JS
- Subtle CSS-only float animations on the mascots

See [spec.md](./spec.md) and [DESIGN_LEARNINGS.md](./DESIGN_LEARNINGS.md).

# POCO POCO — design-stationery-character（日本語）

> **「毎日のお勉強を、ちょっとたのしく。」**
